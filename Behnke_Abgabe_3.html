Create a map of your country of residence with the following features:

  1. a raster background map showing elevation data,
elev <- getData("worldclim", var="alt", res=10)
  1. a vector map showing the country boundary,
ger <- getData('GADM', country='DEU', level=0)
  1. use the vector map to clip the raster map,
crop_elev <- crop(elev, extent(ger))
clip_elev <- mask(crop_elev, ger)
plot(clip_elev)
plot(ger, add=T)
